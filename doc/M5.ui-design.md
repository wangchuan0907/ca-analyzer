## 1. 概述

### 1.1 设计目标
为 ca-analyzer 色度分析仪提供专业、科技感十足的 UI 界面，支持深色/浅色主题切换，参数布局一行两列，整体风格简洁现代。

### 1.2 面板规格
- **主窗口分辨率**：1024 × 768
- **布局方式**：左右两栏（左右各约 480px）
- **默认主题**：深色模式

## 5. 控件样式

### 5.1 输入框（SpinBox / LineEdit）

**深色模式：**
- 背景：`#2D2D44`
- 边框：`1px solid #3D3D5C`
- 圆角：6px
- 内边距：8px 12px
- 聚焦边框：`#6C7EE1`
- 字体颜色：`#FFFFFF`

**浅色模式：**
- 背景：`#FFFFFF`
- 边框：`1px solid #E0E0E8`
- 圆角：6px
- 内边距：8px 12px
- 聚焦边框：`#4A56E2`
- 字体颜色：`#1A1A2E`

### 5.2 单选按钮（RadioButton）

- 圆形选择器，直径 16px
- 未选中：边框 `#6B6B80`，填充透明
- 选中：边框 `#6C7EE1`（深色）/`#4A56E2`（浅色），填充对应强调色
- 标签在右侧，间距 8px
- **顺序：白色（默认选中）、红色、绿色、蓝色**

### 5.3 按钮

#### 主按钮（开始测量）
- 背景：`#6C7EE1`（深色）/ `#4A56E2`（浅色）
- 悬停：`#7C8EF1`（深色）/ `#5A66F2`（浅色）
- 禁用：`#3D3D5C`（深色）/ `#E0E0E8`（浅色），文字 `50%` 透明度
- 圆角：8px
- 内边距：12px 24px
- 文字：`#FFFFFF`，14px，600

#### 次按钮（中止）
- 背景：`#F44336`（启用）
- 悬停：`#E53935`
- 禁用：`#3D3D5C`（深色）/ `#E0E0E8`（浅色）
- 圆角：8px
- 内边距：12px 24px
- 文字：`#FFFFFF`，14px，600

### 5.4 主题切换按钮
- 图标按钮，位于标题栏右侧
- 深色模式显示：☀️（切换到浅色）
- 浅色模式显示：🌙（切换到深色）
- 尺寸：32px × 32px
- 背景：透明，悬停 `rgba(255,255,255,0.1)`

### 5.5 日志区域
- 背景：`#16162A`（深色）/ `#EBEBF0`（浅色）
- 边框：`1px solid #2A2A3E`（深色）/ `#E0E0E8`（浅色）
- 圆角：6px
- 字体：Consolas 12px
- 行高：1.6
- 自动滚动到底部
- 只读，不可编辑

### 5.6 状态显示区
- 进度条：高度 6px，圆角 3px，背景 `#2D2D44`，填充 `#6C7EE1`
- 状态文字：`就绪 / 测量中 / 已中止 / 错误`

## 6. 交互规则

### 6.1 参数修改
- 任意参数修改 → 打屏窗口实时更新（颜色、尺寸、位置）

### 6.2 颜色切换
- 默认选中：白色
- 点击切换 → 打屏窗口立即更新为新颜色 + 当前灰阶

### 6.3 测量流程控件状态

| 阶段 | 参数输入 | 测量按钮 | 中止按钮 |
|------|----------|----------|----------|
| 就绪 | ✅ 启用 | ✅ 启用 | ❌ 禁用 |
| 测量中 | ❌ 禁用 | ❌ 禁用 | ✅ 启用 |
| 结束/中止 | ✅ 启用 | ✅ 启用 | ❌ 禁用 |

### 6.4 灰阶默认值
- 启动后默认显示最大灰阶 **255** 的打屏颜色
- 开始灰阶：0
- 结束灰阶：255

### 6.5 窗口控制
- 主窗口：1024 × 768，固定大小，不可调整
- 打屏窗口：独立窗口，Always on Top，可拖动
- 启动时隐藏 CMD 命令行窗口（`pythonw.exe` 或 `pyinstaller --noconsole`）

### 6.6 重复启动检测
- 检测到软件已运行 → 提示「软件已启动」，退出新实例

## 7. 技术实现要点

### 7.1 UI 框架
- PySide6（推荐）或 PyQt5

### 7.2 隐藏 CMD 窗口
```bash
# 方案一：使用 pythonw.exe 运行
pythonw.exe main.py

# 方案二：pyinstaller 打包
pyinstaller --noconsole --onefile main.spec
```

### 7.3 重复启动检测
```python
from win32event import OpenMutex, WaitForSingleObject
from win32process import GetCurrentProcessId

mutex = OpenMutex(SYNCHRONIZE, False, "ca-analyzer-single-instance")
if WaitForSingleObject(mutex, 0) != WAIT_TIMEOUT:
    # 已存在实例
    QMessageBox.warning(None, "提示", "软件已启动")
    sys.exit(0)
```

### 7.4 打屏窗口
- 使用 Pygame：`pygame.display.set_mode((w, h), pygame.NOFRAME | pygame.SRCALPHA)`
- Always on Top：通过 `SetWindowPos` + `HWND_TOPMOST` 实现

### 7.5 打包图标
- pyinstaller spec 配置中添加 `icon='resources/icons/logo.ico'`
- 确保图标文件存在且格式正确（256×256 PNG 转 ICO）
