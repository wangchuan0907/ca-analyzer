# M5 色度仪测试模块 部署文档

## 1. 打包命令

```bash
cd c:/coder/app/ca-analyzer
pyinstaller --onefile --name ca-analyzer --icon resources/icons/logo.ico --hidden-import pygame main.py
```

**注意：** 当前系统使用 Python 3.14，pygame 官方包无预编译 wheel，需使用 `pygame-ce`（社区版），并通过 `--hidden-import pygame` 显式告知 PyInstaller 打包 pygame。

## 2. 产出物路径

| 文件 | 路径 |
|------|------|
| 可执行文件 | `c:/coder/app/ca-analyzer/dist/ca-analyzer.exe` |
| 图标 | `c:/coder/app/ca-analyzer/resources/icons/logo.ico` |
| SPEC 文件 | `c:/coder/app/ca-analyzer/ca-analyzer.spec` |

**文件大小：** 约 62 MB（单文件，含 PySide6 + pygame-ce + Python 运行时）

## 3. 验证结果

- ✅ 双击运行，GUI 正常启动（主窗口 + 显示窗口同时出现）

## 4. 使用说明

### 运行方式

1. **直接双击运行**：`dist/ca-analyzer.exe`
2. **命令行运行**：
   ```bash
   cd dist
   ca-analyzer.exe
   ```

### 前提条件

- Windows 10/11 x64
- 无需安装 Python（exe 为单文件独立运行）
- 无需色度仪连接即可启动 GUI

### 功能说明

1. **主窗口**（1024×768）：
   - 配置打屏参数（起始坐标、宽高）
   - 配置灰阶参数（开始/结束灰阶）
   - 选择颜色（R/G/B/白）
   - 开始测量 / 中止按钮
   - 状态日志显示区

2. **显示窗口**（独立、始终置顶）：
   - 根据参数实时显示色块预览
   - 由 pygame 渲染

## 5. 打包注意事项

### pygame 兼容性

| Python 版本 | pygame 官方 | pygame-ce（社区版） |
|-------------|------------|-------------------|
| 3.12 | ✅ 有 wheel | ✅ 有 wheel |
| 3.13 | ⚠️ 部分支持 | ✅ 有 wheel |
| 3.14 | ❌ 无 wheel | ✅ 有 wheel |

当前系统为 Python 3.14，使用 `pygame-ce` + `--hidden-import pygame` 解决。
