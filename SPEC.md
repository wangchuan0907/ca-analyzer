# CA-Analyzer 软件规格说明书

## 1. 软件概述

| 项目 | 说明 |
|------|------|
| 软件名称 | CA-Analyzer |
| 软件类型 | 桌面工具（Windows） |
| 核心功能 | 通过串口协议控制设备采集屏幕颜色数据，支持坐标范围与通道范围配置，数据导出为 CSV |
| 目标用户 | 硬件调试工程师、色彩分析测试人员 |

---

## 2. 窗口架构

### 2.1 控制窗口（Main Window）

- **尺寸**：1024 × 768 像素
- **置顶**：否（可被其他窗口遮挡）
- **关闭**：可关闭（关闭时退出整个应用）
- **功能**：参数配置、串口连接、测量控制、数据显示

### 2.2 打屏窗口（Display Window）

- **尺寸**：等于测量区域（Width × Height）
- **置顶**：是（始终置顶，不被遮挡）
- **推拽**：禁止移动
- **关闭**：禁止关闭（跟随主窗口生命周期）
- **功能**：实时显示采集区域的屏幕图像，作为测量参照

## 7. 测量流程（状态机）

```
+------------+    connect     +-----------+
|   IDLE     | -------------> |  CONNECTED |
+------------+                +-----------+
                              | disconnect |
+------------+    start       +-----------+
|  CONNECTED | -------------> |  RUNNING   |
+------------+                +-----------+
                              |  stop      |
+------------+    resume      +-----------+
|   PAUSED   | <------------  |  RUNNING   |
+------------+                +-----------+
                              |  pause     |
+------------+    stop        +-----------+
|  RUNNING   | -------------> |  COMPLETED |
|  /PAUSED   |                +-----------+
+------------+                |  export   |
      ^                       +-----------+
      |                             |
      |        reset                |
      +----------------------------+
```

### 7.1 状态说明

| 状态 | 说明 |
|------|------|
| IDLE | 初始状态，未连接串口 |
| CONNECTED | 串口已连接，等待开始测量 |
| RUNNING | 测量进行中 |
| PAUSED | 测量暂停 |
| COMPLETED | 测量完成，可导出数据 |

### 7.2 状态转换条件

| 转换 | 触发 |
|------|------|
| IDLE → CONNECTED | 用户点击连接，串口打开成功 |
| CONNECTED → IDLE | 用户点击断开连接 |
| CONNECTED → RUNNING | 用户点击开始测量 |
| RUNNING → PAUSED | 用户点击暂停 |
| PAUSED → RUNNING | 用户点击继续 |
| RUNNING → COMPLETED | 测量达到 End 上限，或用户点击停止 |
| PAUSED → COMPLETED | 用户点击停止 |
| COMPLETED → RUNNING | 用户点击重新测量（reset） |

## 8. CSV 导出格式

### 8.1 文件命名

```
ca_data_YYYYMMDD_HHMMSS.csv
```

### 8.2 字段定义

| 列名 | 类型 | 说明 |
|------|------|------|
| timestamp | ISO 8601 | 测量时间戳 |
| index | int | 测量序号（从 0 开始） |
| x | int | 像素 X 坐标 |
| y | int | 像素 Y 坐标 |
| r | int | 红色通道值 (0-255) |
| g | int | 绿色通道值 (0-255) |
| b | int | 蓝色通道值 (0-255) |
| gray | int | 灰度值 (0-255)，公式: 0.299*R + 0.587*G + 0.114*B |
| region | str | 区域标识，如 "0_0_512_512" |

### 8.3 示例

```csv
timestamp,index,x,y,r,g,b,gray,region
2026-04-17T15:30:00.000,0,0,0,255,128,64,151,0_0_512_512
2026-04-17T15:30:00.100,1,1,0,254,129,65,152,0_0_512_512
2026-04-17T15:30:00.200,2,0,1,255,128,63,150,0_0_512_512
```

## 9. 模块设计

### 9.1 src/serial_protocol.py

- 串口参数配置与连接管理
- 协议封装与解析（字节流编解码）
- 错误处理（超时、掉线）

### 9.2 src/measurement_controller.py

- 测量状态机实现
- 屏幕区域截图与颜色提取
- 测量数据缓存管理

### 9.3 src/csv_exporter.py

- CSV 文件写入
- 文件命名与目录管理
- 数据格式化

### 9.4 src/display_window.py

- pygame 窗口初始化
- 图像渲染（实时屏幕区域）
- 窗口置顶与不可移动控制

## 10. 打包配置

```
pyinstaller --onefile --windowed \
  --icon=resources/icons/logo.ico \
  --name=CA-Analyzer \
  --add-data=resources;resources \
  main.py
```

输出：`dist/CA-Analyzer.exe`

## 11. 错误处理

| 场景 | 处理方式 |
|------|----------|
| 串口打开失败 | 弹窗提示错误信息，保持 IDLE 状态 |
| 串口断开 | 自动切换到 IDLE，弹窗提示 |
| 测量超时 | 记录超时日志，继续等待或允许用户中断 |
| CSV 写入失败 | 弹窗提示，检查磁盘空间和权限 |
| 屏幕区域超出范围 | 自动裁剪到屏幕边界，提示用户 |
