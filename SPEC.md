# CA-410 色度分析仪 — 软件规格说明书

## 1. 软件概述

| 项目 | 说明 |
|------|------|
| 软件名称 | CA-410 色度分析仪 |
| 软件类型 | Windows 桌面工具 |
| 核心功能 | 通过串口协议控制色度仪，按灰阶/颜色通道循环打屏并测量 xy/Lv 数据，最终导出 CSV |
| 目标用户 | 显示屏调校工程师、色彩分析测试人员 |

## 7. 测量流程（状态机）

```
IDLE ──→ CONNECTING ──→ CONNECTED ──→ MEASURING ──→ COMPLETED
  │           ↓              ↓             ↓
  │        ERROR           ERROR         ERROR
  │           ↓              ↓             ↓
  └───────────┴──────────────┴─────────────┘
                   （回到 IDLE）
```

| 状态 | 说明 |
|------|------|
| IDLE | 初始状态，未连接串口，所有参数可编辑 |
| CONNECTING | 正在扫描串口并连接 |
| CONNECTED | 串口已连接，等待开始测量 |
| MEASURING | 按灰阶循环打屏+测量中 |
| COMPLETED | 测量完成，已导出 CSV |
| ERROR | 发生错误，弹窗提示后回到 IDLE |

## 8. 串口通讯协议

### 8.1 设备识别

扫描串口，匹配关键词：`"Measuring Instruments"` 或 `"USB 串行设备"`

### 8.2 串口参数

| 参数 | 值 |
|------|----|
| 波特率 | 9600 |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验位 | None |
| 分隔符 | `\r`（回车符） |

### 8.3 命令与响应

| 命令 | 说明 | 成功响应 | 失败响应 |
|------|------|----------|----------|
| `COM,1\r` | 开启设备通讯 | `OK00` | 非 `OK00` |
| `ZRC\r` | 校准命令 | `OK00` | 非 `OK00` |
| `MES,1\r` | 执行一次测量 | `OK00,P1,0,x,y,Lv,...` | 非 `OK00` |
| `COM,0\r` | 关闭设备通讯 | `OK00` | 非 `OK00` |

### 8.4 测量响应解析

响应格式：`OK00,P1,0,0.3274345,0.4191236,4.8075729,0.39,2.1047971`

| 字段位置 | 含义 |
|----------|------|
| index=0 | `OK00` 状态码 |
| index=3 | x 色度坐标 |
| index=4 | y 色度坐标 |
| index=5 | Lv 亮度值 |

## 9. 测量流程详细步骤

1. **点击【开始测量】**
   - 禁用所有参数输入框和按钮
   - 启用【停止】按钮（红色）
   - 记录开始时间

2. **扫描串口**
   - 遍历可用串口，匹配关键词
   - 找不到 → 弹窗报错「色度仪未连接」，回到 IDLE

3. **开启设备通讯**
   - 发送 `COM,1\r`
   - 回复不是 `OK00` → 弹窗报错「色度仪通讯失败」，回到 IDLE

4. **校准**
   - 发送 `ZRC\r`
   - 回复不是 `OK00` → 弹窗报错「色度仪校准失败」，回到 IDLE

5. **循环打屏+测量（Start → End）**
   - 计算当前灰阶 g
   - 根据颜色模式设置打屏窗口背景色 RGB
   - sleep 100ms
   - 发送 `MES,1\r`
   - 解析响应，提取 x, y, Lv
   - 异常 → 弹窗报错「第 N 灰阶测量失败」，跳转到步骤 7
   - 更新右侧面板：实时预览色块、当前灰阶、进度百分比、数据表格

6. **关闭设备**
   - 发送 `COM,0\r`
   - 回复不是 `OK00` → 弹窗报错「色度仪关闭失败」（不中止流程）

7. **导出 CSV**
   - 弹出另存为对话框
   - 无论成功中止还是正常完成，均导出已测数据

8. **恢复界面状态**
   - 参数输入恢复可用
   - 【停止】按钮禁用
   - 进入 COMPLETED 状态

## 10. CSV 导出格式

### 10.1 文件命名

```
色度测量-{起始灰阶}-{结束灰阶}-{YYYYMMDD_HHMMSS}.csv
```

示例：`色度测量-0-255-20260417_153000.csv`

### 10.2 字段定义

| 列名 | 类型 | 说明 |
|------|------|------|
| 灰阶 | int | 灰阶值 0~255 |
| x | float | CIE x 色度坐标 |
| y | float | CIE y 色度坐标 |
| Lv | float | 亮度值 (cd/m²) |

### 10.3 示例

```csv
灰阶,x,y,Lv
0,0.3274,0.4191,4.8076
1,0.3275,0.4190,4.8080
2,0.3276,0.4189,4.8085
...
255,0.3280,0.4195,5.1000
```

## 12. 错误处理

| 场景 | 处理方式 |
|------|----------|
| 色度仪未连接 | 弹窗「色度仪未连接，请检查设备连接」 |
| 通讯开启失败 | 弹窗「色度仪通讯失败」 |
| 校准失败 | 弹窗「色度仪校准失败」 |
| 测量异常 | 弹窗「第 N 灰阶测量失败」 |
| 设备关闭失败 | 弹窗「色度仪关闭失败」（不中止导出） |
| CSV 写入失败 | 弹窗「CSV 导出失败，请检查磁盘空间和权限」 |

## 14. Tauri 配置要点

### 14.1 tauri.conf.json 关键配置

```json
{
  "productName": "CA-410 色度分析仪",
  "identifier": "com.ca-analyzer.app",
  "build": {
    "devtools": true
  },
  "app": {
    "windows": [
      {
        "title": "CA-410 色度分析仪",
        "width": 1024,
        "height": 768,
        "resizable": false,
        "center": true
      }
    ]
  },
  "bundle": {
    "icon": ["icons/logo.ico"],
    "target": ["msi", "nsis"],
    "windows": {
      "nsis": {
        "installMode": "currentUser"
      }
    }
  }
}
```

### 14.2 打屏窗口创建（Rust 侧）

```rust
// 在测量开始时创建打屏窗口
let display = WebviewWindow::new(
    &app.handle(),
    "display",
    WebviewUrl::App("display.html".into()),
    WebviewWindowBuilder::new(app.handle(), "display", WebviewUrl::App("display.html".into()))
        .position(x as f64, y as f64)
        .inner_size(width as f64, height as f64)
        .always_on_top(true)
        .decorations(false)
        .resizable(false)
        .closable(false)
        .build()
);

// 测量过程中更新背景色
display.eval(&format!("document.body.style.backgroundColor = 'rgb({},{},{})';", r, g, b));
```

### 14.3 tauri-plugin-serial 权限

在 `tauri.conf.json` capabilities 中添加：
```json
{
  "identifier": "serial",
  "allow": [{ "serial": { "all": true } }]
}
```
